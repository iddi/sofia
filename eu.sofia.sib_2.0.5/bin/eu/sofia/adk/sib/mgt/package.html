Provides the classes that manages the creation of new SIB instances. Only one SIB OSGi bundle can be initiated at a time, but one SIB bundle can contain multiple instances of the Semantic Information Broker (SIB), so that the user/developer can use it for interaction with the current Smart Applications.

Package Specification

The package provides the methods to get the information about current started SIB-instances and how to create, start, stop and destroy those instances.
